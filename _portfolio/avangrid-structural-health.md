Field validation is a core component of structural health monitoring (SHM). This project describes a site campaign at the Avangrid Test Site, where we deployed telemetry systems to monitor real-time strain on operating wind turbine blades.

By pairing structural strain sensors with atmospheric LIDAR units, we collected a multi-modal dataset mapping atmospheric load conditions directly to structural stresses. The findings demonstrate a path toward predictive maintenance, showing a 12% improvement in forecasting model accuracy.
